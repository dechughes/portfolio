# **Page 4 — Measure Creation**

With the dataset created, imported, and linked in Power BI, the next step is to build measures that calculate and display the key commercial metrics. These measures allow the business to understand performance, diagnose friction points, and analyse user behaviour across the lifecycle.

To structure this effectively, four dashboards were created, each reflecting a core analytical need in a real business environment.

---

## **Executive Overview Dashboard**

This dashboard focuses on the high‑level health of the business, including:

- Growth trends  
- Churn behaviour  
- Revenue channels  
- Overall performance indicators  

It provides a top‑down view suitable for leadership and strategic decision‑making.

---

## **Funnel Analysis Dashboard**

This dashboard examines the full user journey to identify where users drop off and how efficiently they progress through the funnel. It highlights:

- Activation rates  
- Conversion rates  
- Churn rates  
- Channel effectiveness  
- Friction points in the user journey  

This enables diagnosis of where improvements in onboarding, product experience, or acquisition quality are needed.

---

## **Experiment Analysis (A/B Testing) Dashboard**

This dashboard compares the performance of experiment groups A and B to understand:

- Which group performs better  
- How behaviour differs across the lifecycle  
- Revenue differences between groups  
- Whether the experiment meaningfully impacts key metrics  

This supports data‑driven decision‑making around product changes and feature rollouts.

---

## **Cohort Retention Dashboard**

This dashboard focuses on retention behaviour over time, answering questions such as:

- Which users stay on the platform?  
- Does churn stabilise after a certain period?  
- Do different signup cohorts behave differently?  
- Are certain acquisition periods more or less sticky?  

This enables long‑term behavioural analysis and helps identify structural retention issues.
